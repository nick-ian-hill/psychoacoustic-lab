# Psychoacoustic Lab

A system for psychoacoustic experiments and measurements.

## Project Structure

- `web-app`: The frontend application for running experiments.
- `mcp-server`: Model Context Protocol server for integrating with AI tools.
- `shared`: Common schemas and logic.
- `examples`: Usage examples and test scripts.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the web app:
   ```bash
   cd web-app
   npm run dev
   ```
